# IOTAIR

## Getting started

```bash
yarn
yarn start
```

## Project structure

* config/default.yml: configuration settings
* res/: static resources like images, fonts and webpages
* src/: project source code. Entry point is index.js
* Dockerfile.template: configuration of the dockerized environment that will run on Balena OS
